# Skill: QA Testing

## Purpose
Generate and run unit/integration tests for a given module using a sub-agent.

## Checklist
- [ ] Identify all public functions/methods in the target file
- [ ] Write unit tests for happy path
- [ ] Write unit tests for edge cases and error states
- [ ] Mock external dependencies (DB, APIs)
- [ ] Run tests and confirm all pass
- [ ] Report coverage percentage

## Usage
Invoke with: "Run QA tests on [file or module]"
Sub-agent model: Sonnet
